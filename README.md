
# ThinkMoney - HT6 Submission




## Run Locally

Clone the project

```bash
  git clone https://github.com/melhemfares/ht6-2022.git
```

Install dependencies

```bash
  npm install
```

Run the backend server

```bash
  cd api -> nodemon start
```

Run the client server

```bash
  cd client -> npm start --web
```
